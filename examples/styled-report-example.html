Styled Report Example

A more realistic document for testing HTML-to-PDF export quality.

This sample uses backgrounds, spacing, and a two-column layout to show that the exporter preserves styled HTML while still suppressing leaked browser print chrome.

What should not appear

  • Timestamp headers
  • Page title headers
  • Local file paths in footers

What should appear

  • Document backgrounds
  • Typography and spacing
  • Layout and content only
